# 2_4_5_Logical_Operators.py

Given an int n, return the absolute difference between n and 21, except return double the absolute difference if n is over 21.

### Sample Run 1
```text
Enter a number: 19
2
```

### Sample Run 2
```text
Enter a number: 10
11
```

### Sample Run 3
```text
Enter a number: 21
0
```
