# 2_5_3_processing_strings.py


Given a string, output a new string made of 3 copies of the last 2 characters of the original string. The string length will be at least 2.

### Examples

```text
Enter a word: Hello
lololo
```

```text
Enter a word: ab
ababab
```

```text
Enter a word: Goodbye
yeyeye
```
