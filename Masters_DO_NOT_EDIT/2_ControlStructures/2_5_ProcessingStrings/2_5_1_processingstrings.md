# 2_5_1_processing_strings.py

Given a string name, e.g. "Bob", output a greeting of the form "Hello Bob!".

### Examples
hello_name('Bob') → 'Hello Bob!'
hello_name('Alice') → 'Hello Alice!'
hello_name('X') → 'Hello X!
```text
Enter a name: Bob
Hello Bob!
```

```text
Enter a name: Alice
Hello Alice!
```


```text
Enter a name: X
Hello X!
```
